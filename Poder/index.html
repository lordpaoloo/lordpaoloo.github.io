Card Text

Poder is an advanced data extraction tool designed to automate and simplify Facebook scraping. Whether you're conducting research, monitoring trends, or gathering insights, Poder delivers fast, accurate, and efficient results with minimal effort.

With its multi-threaded engine, real-time progress tracking, and modern PyQt5 interface, Poder makes data collection seamless and user-friendly.

Key Features


  • -Facebook Page & Hashtag Search – Easily extract data by name or hashtag.
  • -Auto-Scraping Mode – Continuous data extraction without manual input.
  • -Multi-Threaded Scraping – Faster performance for large-scale data collection.
  • -Real-Time Logs & Progress Tracking – Stay updated with live logs.
  • -Structured Data Storage – Save and organize extracted data for analysis.
